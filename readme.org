* Synopsis
Even though this is technically a school project, I am mostly using this as an excuse to push myself to finally develop
the little game I always wanted to make and get better-ish at programming. This project is public for ease-of-access,
but is not intended to be viewed by a wide audience. If a stranger does somehow stumble across this, hello, feel free
to look at the source code. No guarantee it will be good. If all goes well, this will a nostalgia project in a few years.
